2019 年中（学期）总结

发表于 2019-06-13 | 更新于 2019-06-16 | 分类于 个人成长 | 阅读次数：
本文字数： 1.7k | 阅读时长 ≈ 2 分钟

下午考完试，一本科韩国同学对我说：“第二学期了吧，快一年了，恭喜你。”

时间在奔跑，转眼已经研究生第二学期结束了。下面对这半年的一些事情进行复盘和总结。

别问我为什么这么早写年中总结，刚好期末结束有点点时间而已。

课业学习

1，2 月是假期，这两个月主要集中提升代码能力，学习 Python 进阶内容，并且一天一道 leetcode 算法题，然而这在 3 月之后就没有坚持下去了。

3 月开课后，选了 C 语言编程课，数字图像处理和机器学习特论（TensorFlow）。 机器学习这门课还比较容易都是之前学过的内容，教授人也好，没作业压力。而数字图像处理这门课，作业就比较多了，几乎每两周一个作业，作业的主要内容是看论文，然后实现并比较不同的算法。不难，但是很耗费时间，需要花精力看论文，搞懂算法，实现再比较结果。开学后，在简书更新的大多文章都是和数字图像处理有关，就是因为这个原因。不过也挺感谢这位教授的，不仅看了很多论文，还增强了代码能力。

至于 C 语言这门课程对我就比较有挑战了 ，本科时就学的不好，现在更没底了，还好前期有实验室越南博士的帮助与鼓励，很顺利的完成了作业和期中考。期中考之后 C 语言的课程内容转向了强化学习，主要是为了期末的 Term Project 铺垫基础知识。还好最后教授降低了要求，给了 Python 代码，要求修改成 C 语言代码并实现可视化就可以。

4 月中旬，也就是期中考考试周的时候，和教授算是定下了一个课题。5 月教授要求动手从头实现网络的代码，而不是依赖现有的框架，并给了他之前的代码给我参考。目前这块的代码还在努力实现中。

总之，这学期过得蛮紧张又蛮幸运的。

理财入门

本科以来的财务管理习惯实在不好，今年下定决心痛改全非，努力做好自己的开支和预算管理。学习理财主要是学习了少数派的两个课程，如下图所示（非广告 x）一个是个人记账管理；一个是理财投资新手的操作指南。

看完课程之后也一直关注也谈钱大大的微信公众号，从他的 GTB （Getting Things Bought）系列文章中学到了很多，强烈推荐大家去看。3 月开始用 MoneyWiz 这个软件进行记账和预算管理，目前还处于不断的摸索和磨合中。

读书看剧

年初定下了看 3 本非专业外的书的计划。目前已经看了 2 本，一本是阮一峰老师的《未来世界的幸存者》，当时看完这本书的感觉和这几天刷某不知名的英剧的感觉一样，即对未来科技的高速发展感到兴奋，同时又对未来世界感到悲观。科技带来进步的同时，可能也会引起某些事物的倒退。另一本是 《富爸爸穷爸爸》，理财入门必看书单之一。

这段时间花在看剧上的时间蛮多的。晚饭时间刷 《King of Hill》，竟然刷完了 12 季，除了第 9 季没有资源外，其他都刷了一遍。

5 月也抽空看了自己感兴趣的两部韩国电影。一部是轻松愉快为主的 《女警察》，一部是有名的《寄生虫》。在看《寄生虫》之前没看任何国内和国外的剧透 ，单纯以为和《汉江怪兽》一样的类型，看完之后震撼很大，也很难过。立马想到阮一峰老师书中提到的 「穷忙族」，一生都困在穷忙之中，这是多么不幸啊。

复盘反思

年初脑热定下的计划有很多不合理的地方，这次决定调整并减少计划，使其符合实际，也更容易被执行。

  • 加强代码能力 - 目前在努力中
  • 提高英语口语和阅读能力 - 看论文不吃力，能进行日常简单交流
  • 简书或者博客总文字达到 25 万字
  • 完善自己的个人博客搭建 - 界面优化，搬运简书文章
  • 至少看 3 本专业之外的书 – 已看完两本
  • 看完一些遗留的付费课程，并进行总结输出
  • LeetCode 争取刷完 Easy 部分
  • 发一篇小论文
  • 学会一点理财的知识
  • 少上微博
  • 作息规律 - 目前算是达到了
  • 多运动，少吃零食 - 目前在努力中

附上之前的版本：

去掉了提高韩语那一条，因为规划了一下，自己以后不会留在韩国，所以英语的作用比韩语来得更重要。还去掉了 Docker， Linux，Kaggle 这三条，因为目前这三条的优先级不是特别高，也没有那么多时间分配出来。

修改之后总共 12 条 flag，已划掉 2 条，目前努力在做的有 4 条。希望年末再来拔的时候，可以划去更多条。
