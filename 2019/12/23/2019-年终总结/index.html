2019 年终总结

发表于 2019-12-23 | 分类于 个人成长 | 阅读次数：
本文字数： 2.3k | 阅读时长 ≈ 2 分钟

去年开始学着总结，学着写年中，年终总结。今年总觉得动力不足，可能和今年整年都过得比较丧有关吧，没什么值得记录的事情。不过，不论如何，又是一年到头了，不管好坏，不管大事小事，都值得记录。

2018 计划复盘

年初定下了挺多计划，在年中的时候进行了一些调整，现在再来复盘一下。

  • 加强代码能力 - ✔
    • 感觉今年代码能力提升挺多的。一整年都没少码代码。3 月份选的图像处理课程，每个作业都需要看论文然后实现论文算法，一开始觉得挺难，每次都耗费很多时间。下半年选 3D 视觉的时候就习以为常了。
    • 5 月份开始动手从头实现一个网络的前向和反向传播，中间 debug 了很久都没达到预期结果，直到 8 月底才有了结果。现在把 NumPy 和 PyTorch 结合起来写了一个网络层，可以在 Pytorch 中调用，不过训练时间相较于 CNN 要花费很多，这个之后要想办法进行优化。
  • 提高英语口语和阅读能力 - ✔
    • 英文阅读这个一直有在做，因为要看论文。
    • 口语这个，是在报考雅思之后才有了动力去做。雅思开课后到考试周之前，每天都会进行半小时左右的晨读，不过在考试周搁置了，之后还会继续练习的。晨读的效果不错，感觉最近和越南博士对话的时候，会说较长的句子了。
  • 至少看 3 本专业之外的书 - ✔
    • 《未来世界的幸存者》- 这本书是阮一峰老师写的，是一本关于未来思考的书。
    • 《富爸爸穷爸爸》 - 年初学习理财时，看的一本入门的书。
    • 《我有一只叫抑郁症的黑狗》- 抑郁症科普读物
    • 《永久记录》- 斯诺登写的自传，感觉可以在年前看完。
  • 简书或者博客的总文字达到 25 万字 - ×
    • 太看得起自己了，下半年开始就没怎么输出了。
  • 看完一些遗留的付费课程 - ×
    • 依旧没多少时间看，最近在看的付费课程是雅思。
  • LeetCode 争取刷完 Easy 部分 - ×
    • 忙于自己的课题和专业的作业，没多少时间学习算法。
  • 少上微博 - ×
    • 下半年开始想吐槽的事儿太多了，以及开始关注了一些明星。
  • 作息规律 - ×
    • 考试周熬了两个通宵，最近才开始慢慢恢复规律。
  • 多运动，少吃零食 - ×
    • 少吃零食做到了，多运动没做到，感觉最近又胖了。
  • 发一篇小论文 - ×
    • 这个估计得推迟了，实验的结果一直不是很理想。

现在看来，接下来做 2020 年计划的时候，要好好规划和思考了。

2019 看了什么

1 月份开始看了 NHK 的纪录片后，一发不可收拾，感觉纪录片也蛮有趣。今年看了挺多的：（写了评语的，都是我个人会比较推荐給别人的）

  • AlphaGo - 对科技和 AlphaGo 背后相关故事感兴趣的，值得一看。纪录了李世石和初版 AlphaGo 对局的一些故事。
  • 三和大神（NHK）- 看完，第一次知道了 “三和大神” 这一特殊群体。他们以日当为主要收入，干一天玩三天，只为当下而活。
  • 徒手攀岩
  • 改革开放帮助中国的日本人（NHK）
  • 美国工厂
  • 光棍儿（NHK）

今年也挺奇怪看电视剧竟然看得比电影还多：

  • 美剧《切尔诺贝利》- 五星推荐，谨以此纪念那些受难和牺牲的人。剧里展现的一些辐射反应还是比较还原的。
  • 英剧《nian nian》 - 五星推荐，不知道未来会怎样，年复一年的变差吗。当生活不再，活着或许就是唯一的期望。
  • 美国成人动画《The King of Hill》（中译：一家之主）- 五星推荐，非常适合下饭，一集 20 分钟，正好是一顿饭的时间。讲了美国德州阿伦小镇上四位好基友及其家庭中的故事。关于对孩子的一些教育问题会有些启示。也非常适合练听力。
  • 美剧《为了全人类》
  • 《陈情令》
  • 《小欢喜》
  • 韩剧《天空之城》
  • 《家有儿女》第一部（重看）
  • 《冰上的尤里》（重看）

电影：

  • 《82 年生的金智英》- 五星推荐，放在最前面也是因为它是一部揭露韩国女性现状的一部电影。平淡，真实又窒息。
  • 《寄生虫》- 五星推荐，看完就觉得特别难过和无助，也有种永无出头之日之感。
  • 《小丑》- 五星推荐，虽然比较丧比较超出三观。
  • 《Swing Kids》- 五星推荐，朝鲜战争为背景，fxxking ideology.
  • 《国王的演讲》
  • 《女警察》
  • 《铁雨》
  • 《驯龙高手》第一部

2019 学了什么

  • 学会了双拼，手机端已经完全熟悉双拼了，速度很可以，就是手残更严重了。
  • 从 TensorFlow + Keras 转向了 PyTorch，import torch as tf ，简单又灵活。
  • 使用现有工具搭建个人网站。
  • 理财入门 - 学会制定预算，记录消费。
  • Presentation，经历过两次比较有压力的发表，现在总算不怯场了。总之多练习是没错的。

2019 吐槽了什么

只可意会，不可言传。就当忘了吐槽了些什么，发生了太多不好的大事儿。

2019 使用的一些 App

  • Notion - 代办事项，学习笔记，协同工作， all in one。
  • MoneyWiz - 记账 App，功能很强大，喜欢预算制定和支出收入分析的功能。
  • IFTTT - 可以自动化一些事情。
  • 后续 - 新闻事件后续跟踪，很少见的一类 App。

2019 使用的一些网站

  • v2ex - way to explore, 程序员相关的一个讨论站。
  • GitNews - git.news，基于 Github 上的热门项目做的一个排行。
  • paperwithcode - 收集深度学习相关领域的一些 state-of-the-art，以及提供的论文会有配套的代码。
  • whereisscihub - 获得 sci-hub 有效链接的一个网站，sci-hub 可以免费下载一些学术论文。
