2018 年终总结

发表于 2018-12-17 | 更新于 2019-01-28 | 分类于 个人成长
本文字数： 1.7k | 阅读时长 ≈ 2 分钟

写在前面

早上起来 Windows 软件提醒 2018 已经完成了 96% 了。想想是时候该写年终总结了，也正好现在复习不下去了。之前年中的时候，在知识星球写了一篇年中总结，没想到现在已经年终了，可真快。相对来说，2018 这一年对于我也是比较有转折的一年。

2018 年的总结和 2019 年的愿景可以概括为以下这几句话，这几句话已经发在掘金的沸点上了：

毕业

今年的头等大事主要是毕业。从 2 月底寒假结束，坐着飞机从西北到东南上学，就一直焦虑着。为毕业设计和论文焦虑。当时的毕业设计才刚开始完成一个 demo，还有很多东西没做好，但是快要邻近答辩了。那会每天都是吃也吃不好，睡也睡不好，刷着手机都带着点罪恶感。好在最后，实在没有头绪，向导师承认 「我个垃圾按他的思路做不出来」后，导师宽容的允许我可以准备论文了。由于之前假期，做项目的时候，边学习边输出，在简书总结了一些文章，写论文正好可以派上用场，基本一星期就完成了论文的编写。最后的答辩，因为选题比较偏，答辩老师也没多为难，就放行了。一辩结束，我就马上回家准备东西，然后直奔杭州开始找工作了。

毕业典礼那几天是端午假期，和同学赶了趟凌晨的火车，回到学校，急急匆匆，忙东忙西。最后也没赶上校长的拨穗留念就赶动车回杭州上班了，这是一个遗憾。

实习

四月中旬来到杭州，暂住在学姐家，每日都投简历面试。大概四月末的时候，收到 3 个 offer，最后选择去往西溪的那家人工智能公司。

公司是一家以语音及智能音响为主的创业公司。报到的当天就从语音前端测试组调到 IOT 组支援了。因为当时，IOT 组业务量增加，急需测试人员。在 IOT 组，主要跟随两位小哥哥做智能音响与智能家居方面的测试，这方面的测试还是挺有趣的。记得第一天成功配上小米的智能台灯，并用音响成功控制后，那欣喜的感觉很不一样，突然就感受到了智能带来的变革。

在 3 个月的实习期间，学到挺多测试方面的知识，也了解到许多关于智能音响的技术。并且在这期间，有幸地参加了公司的第一届开发者大会，在大会期间主要担任志愿者。当自己曾经测试过的「智障」音响，在发布会上成功演示的时候，有一种自家孩子终于长大了的感觉。虽然我只负责 IOT 这一部分的测试，但还是挺激动的。

后来，在自己的考量之后，决定继续读研了。在 8 月 10 号，告别了这家实习了 3 个月的公司，也告别了炎热的杭州。

很感谢当时的前端语音测试组的组长，邀请我加入这家公司。也很感谢 IOT 团队照顾我的小哥哥小姐姐们，他们都是一群年轻、活力，很有想法的人。

读研

四月中旬，也是毕业答辩之后，开始着手准备资料申请韩国的大学。当时的想法是一边找工作一边申请，即使没申请上，也能继续有一份工作做着。

五月底，在截止日一星期前递交了材料，赶上了申请的末班车。6 月底回学校参加毕业典礼的第二天，正好在邮箱发现了导师的邮件。7 月中，正式收到学校的录取通知书。之后，跟老大提了辞职，交接完工作，就回家准备出国的东西了。

9 月 1 号，到达学校，和导师及实验室的小伙伴见面吃饭。初印象是，导师是一个很 nice 的人。第一学期，导师没有给定方向，完全没压力，全靠自己自学。这期间学习了很多 Arduino ，Rapid Miner 和睡眠检测的相关知识，也看了一些自己感兴趣的 papers。明显感觉这段时间的英语阅读能力大大的提升了。

目前在准备第一学期的期末考试，希望能有个好成绩。这样，来年就可以多选一门课。

其他

  • 写作
    在不久之前，自己在简书的总字数达到了 10 万字，算是完成了 2018 年的一大目标。今年也开始在掘金，微信公众号上尝试发布自己的一些文章，也开始尝试自己搭建个人博客。希望自己在写作这条道路上越走越远，也希望能够多多输出一些有价值的文章。

  • GitHub
    2018 年也开始在 GitHub 上发布自己做的一些小项目以及学习相关的一些笔记。也算是符合了年初定下的 「Get hands dirty」的初衷。希望明年继续努力，争取在代码能力上有突破，能够懂得更多的算法，能够复现一些论文的结果。

寄语

无欲无求，不骄不躁，顺其自然，佛系科研。

Arduino 入门项目系列(5) - Android 手机通过蓝牙语音控制 LED

发表于 2018-11-19 | 更新于 2019-01-28 | 分类于 Arduino
本文字数： 3k | 阅读时长 ≈ 3 分钟

为了加快 Arduino 的学习，决定周一这一天的上午也用来学习 Arduino。今天还是继续学习蓝牙模块的使用。实现通过蓝牙模块，在手机端语音控制 LED 的功能。这种思路后续再扩宽的话，加上动手能力强的话，完全自己搭建智能家居系统。

本文章主要参考的是这篇：Android 通过蓝牙语音控制 Arduino，非常感谢作者的教程！

设置蓝牙

本人使用的蓝牙模块是 HC-05，由实验室的小伙伴友情提供。因为是他人使用过的，所以得进行初始化设置。

  • 进入 AT 模式：具体参考这篇文章：Arduino使用HC05蓝牙模块与手机连接
  • AT 模式的一些命令：
    进入 AT 模式后，重新设置名称和密码，并将蓝牙设置为从模式，因为这个项目中，手机是主机。

硬件准备

  • Arduino UNO 一个
  • 蓝牙模块 HC-05 一个
  • 红色和黄色 LED 各一个
  • Android 手机（有谷歌套件）
  • 面包板一块
  • 连接线若干

软件准备

  • Arduino IDE
  • 在 Google APP Store 或者其他安卓市场下载 Arduino Voice Control

电路连接

黄色 LED 的连线：长脚 ——>先接个电阻，再连接 13，短脚 ——> GND
红色 LED 的连线：长脚 ——>先接个电阻，再连接 12，短脚 ——> GND
蓝牙的连线：VCC ——> 5V，GND ——> GND，RXD ——> TXD，TXD ——> RXD，这里需要注意蓝牙的 RXD 要与 Arduino 的 TXD 连接，TXD 要与 Arduino 的 RXD 连接，两者是交错连接的。同时在烧录的时候，注意断开 TXD 和 RXD 的连接，要不然烧录的时候会受到干扰，继而会报错。

实现代码

Arduino_Learning/VoiceControlLed

烧录代码

代码编译完没问题后，点击图中所选按钮将其上传至 Arduino UNO 板上。

手机设置

  1. 点击 CONNECT，选择蓝牙，点击连接。

  2. 点击中间的麦克风，进行语音输入

实际结果

  • 语音输入 turn on the yellow：黄色的灯会打开
  • 语音输入 turn on the red：红色的灯会打开
  • 语音输入 blink：两个灯会一起闪烁
  • 语音输入 stop：两个灯会停止闪烁，并保持常亮
  • 语音输入 turn off：两个灯都会熄灭

这个项目的结果最好的显示方式是录视频，但我一个人忙不过来，就放弃了。结果可以参考上面那篇文章的结果。我主要增加了按颜色实现语音控制 LED ，以及 LED 闪烁功能的开启和关闭这些功能。只是代码方面的修改，及语音方面输入的不同。

心得总结

刚开始在看到一篇用 BitVoice 这个工具实现语音控制的文章，但下载了 BitVoice 这个软件后，没有搞懂怎么使用，就放弃了。后来在收藏夹里又突然发现上面这篇文章，就赶紧参考写代码实现了。在实现的过程中，感觉英文发音比较准的话，会比较容易控制；发音不好的话，就会识别不清，得多次输入。
